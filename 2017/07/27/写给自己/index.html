写给自己

入职已经半个多月了，但是感觉自己还没有进入状态。前段时间偶然在网上看到Sunny（孙源）说iOS进阶的速度：

纯日常开发 < 纯看书、博客 < 自己试验、Demo < 写博客 < 系统性分享和讨论 < 提供完整的开源方案

顿时心里感觉：握了个草！原来我一直停留在进阶速度最慢的两个阶段，难怪呢！
于是又想捡起好久没写的博客，重操旧业。但是感觉以前的博客质量太低了，所以新搭了一个，前后折腾了三四天（净整些没用的Orz）

其中有一天早上我心里就暗想，以后要至少半个月写一篇有质量的博客。但是这个念头刚萌生不超过10秒，我心里就又暗想：半个月够呛啊，要不然还是一个月吧。。。Orz

在这里给未来的自己写下这段话，希望将来的自己如果连一个月一篇有质量的博客都拿不出手的时候，看到这段话产生羞愧之心。。。
